2020.3.20 第二周学习
- HTML 标签通常是成对出现的，比如 "<b>" 和 "</b>"
- 标签对中的第一个标签是开始标签，第二个标签是结束标签

# HTML标签
- 标题
```
<h1>Title</h1>
<h2>Title</h2>
<h6>Title</h6>
```
- 段落
```
<p>和</p>
```
 - 链接
```
  <a>和</a>
<a href="http://baidu.com">百度一下你就知道</a>  //在 href 属性中指定链接的地址。
```
- 图像
```
<img>、</img>
```
# HTML元素
### HTML 元素语法
- HTML 元素以开始标签起始
- HTML 元素以结束标签终止
- 元素的内容是开始标签与结束标签之间的内容
- 某些 HTML 元素具有空内容（empty content）
- 空元素在开始标签中进行关闭（以开始标签的结束而结束）
- 大多数 HTML 元素可拥有属性
